
# Farsaj.az - Turbo.az Clone Project

This project is one of my first experiences in web development. My goal is to replicate the front-end design of the popular Azerbaijani car listing site, **Turbo.az**, to practice and improve my coding skills.

## 🚀 Technologies Used
* **HTML5** - For the structure of the website.
* **CSS3** - For design, layout, and Flexbox styling.
